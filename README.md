# Notepad
Notepad is a lightweight, standalone text editor written in pure HTML, CSS, and JavaScript.

## Install
You have two options: the opener and the source.<br><br>

The opener is a launcher, allowing Notepad to be updated with one click.<br>
To install, download opener.html and follow the instructions.<br>
On Firefox, you may need to allow popup windows the first time. Close the example.com window and open the opener.html again.<br><br>

Alternatively, you can downlaod Notepad.html directly.
